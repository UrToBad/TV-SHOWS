# TV-SHOWS

Projet Web ULCO

## Description

Ce projet est une application web permettant de gérer et de consulter des informations sur des séries.

## Fonctionnalités

- Gestion des séries
- Connexion sécurisée à une base de données SQL via PDO

## Prérequis

- PHP 7.4 ou supérieur
- Serveur web
- Base de données MySQL
